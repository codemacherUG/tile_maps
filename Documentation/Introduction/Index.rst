What does it do?
================

Tile Maps allows you to integrate OpenStreetMap maps into your site in compliance with GDPR and without any third party requests from the client.
It is based on the Tile Proxy extension which realizes a proxy and caching mechanism for OpenStreetMap.
Furthermore, tt_address entries can be output as markers.
